If there is TWAIN support for reflective and transmissive scanning, DTWAIN allows switching between reflective (paper) and transmissive (film/transparency) acquisition by usage of the following functions

 

Enabling Refllective / Transmissive acquisitions

DTWAIN_SetLightPath

 

Testing for support

DTWAIN_IsLightPathSupported

 

Retrieving Current Light Path

DTWAIN_GetLightPath
